# Shared Infrastructure

Cross-engine infrastructure used by two or more engines. Code belongs here only when it serves multiple engines and does not conceptually belong to any single engine.

## Contents
- `consensus/` — Multi-model consensus mechanism (§2.2). Used by atomization, excerpting, taxonomy, and source engines.
- `validation/` — Algorithmic validation tools: schema validation, structural integrity checks (§8.1).
- `feedback/` — Feedback loop infrastructure: correction storage, pattern analysis, regression testing coordination (§8.3).
