- **Before crafting ANY artifact by hand** (prompt, document, plan, handoff, dispatch template, relay message, evaluation rubric), check: is there a specialized tool, skill, or plugin designed for this exact task? Use `/prompt-architect` for prompts. Use `/ce:brainstorm` for feature exploration. Use `/ce:plan` for implementation plans. Use `/ce:review` for code review. The hand-crafted version is ALWAYS the fallback, never the first choice.
- **"Good enough" is not acceptable.** Every artifact that touches the excerpting engine directly affects knowledge quality. A prompt that's 80% optimized means the LLM follows 80% of the rules. That remaining 20% is where silent corruption hides.
- **The check takes 5 seconds. The cost of skipping it is unbounded.** If you're about to write something and a tool exists for it, use the tool first. Always.
