## Classification Rules

| Category | Meaning | Action |
|----------|---------|--------|
| **DUPLICATE** | Same concept already captured in F-series MAQ | Skip — reference the existing MAQ-ID |
| **EXTENDS** | Adds new angle/evidence to existing MAQ/FP | Merge into existing MAQ — note what's new |
| **NOVEL** | Genuinely new concept not in F-series | Assign new MAQ-ID (MAQ-089+) |
| **REINFORCES** | Adds weight but no new information | Note and skip — strengthens existing |
| **META** | Documentation/vocabulary/methodology | No MAQ needed — document in SPEC or reference |
| **DEFERRED** | Cross-engine or future feature | Move to Section J (deferred) |
| **FP-CANDIDATE** | Potential new foundational principle | Needs multi-coworker validation |
