# Raw hadith vs full explanation vs chunk explanation distinction

- **Raw hadith**: the proof text as a proof object.
- **Full explanation**: explanation of the hadith as a whole.
- **Chunk explanation**: explanation tied to local segments inside the hadith.

The owner’s raw reaction explicitly wants all three in the ideal library.

# غريب الحديث as a special explanatory branch

غريب الحديث is not just miscellaneous commentary. It is a specific explanation type focused on words, expressions, lexical meaning, sometimes grammar-like clarification. It deserves branching distinction rather than being collapsed into generic commentary.

# Proof-relationship doctrine pressure

The case pushes beyond single-proof handling. Under one proof leaf, proofs can relate to each other in materially different ways:
- different versions of the same hadith
- multiple proofs completing one ruling together
- apparent contradictions resolved by combination
- قول / فعل / complementary evidence relations

The library should not treat all proof co-presence as the same kind of relation.

# Proof-identifier doctrine pressure

The owner explicitly rejects narrator-name-only identification. `حُمْرانَ` is not a safe unique identifier for a proof branch by itself. A stable identifier must carry enough uniqueness to prevent study confusion.

# Vocabulary / dictionary branching doctrine pressure

The presence of غريب الحديث shows that vocabulary-level explanation is a real layer. That opens a larger opportunity:
- dedicated غريب الحديث branches
- grammar / sarf-style sub-branches
- dictionary ingestion targets that land in the most specific relevant locations

# Cross-topic proof reuse doctrine pressure

The same proof may contain material relevant to more than one topic. If that material remains trapped in only one branch, the owner loses understanding when studying the other branch. The owner explicitly treats this as lost potential. So the case pressures cross-topic reuse handling, not just local packaging.

# Unlimited placement doctrine pressure

The owner’s raw reaction explicitly rejects limiting placement by current science or current branch. If a proof segment belongs elsewhere too, placement should not be artificially capped.
