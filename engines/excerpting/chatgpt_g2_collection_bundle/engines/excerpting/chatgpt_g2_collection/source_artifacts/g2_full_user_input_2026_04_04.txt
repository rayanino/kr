## My raw comments

This actually reminds me of the topic: when the tree gets to, for example: proofs → hadiths → hadith عثمان بن عفان, is there any further granulation?

The answer is: ideally yes. CATCH THIS EXPLANATION (VERY IMPORTANT): a common methodology between scholars is that when explaining a hadith they explain it chunk by chunk. For example:

“عن حُمْرانَ مَوْلَى عُثْمَانَ بْنِ عَفَانَ” → stop → explain this (who was he etc)

“أنه رَأى عُثْمَانَ دَعَا بِوَضوء” → stop → explain this (what is a ‘وَضوء’ etc)

…

Preferably, my ideal structure in the library would be:

… → hadith عثمان بن عفان → raw (this is the part where the hadith ready-to-memorize (externally gathered by the library in correct formatting) lives)
                         → full explanations (this is where non-chunked ‘general’ explanations would come that do not address a specific part of the hadith but rather try to explain the hadith in its entirety)
                         → chunk1 → excerpts explaining this chunk
                         → chunk2 → excerpts explaining this chunk

But obviously, this introduces more complexity. Rarely do two or three follow the exact same chunking choices when explaining hadiths, let alone tens and hundreds. Also: the longer the hadiths, the more complex this operation becomes.

But that is not a challenge too big to solve. The ideal library should be able to do this without a problem.

For the specific excerpt in question, this would mean something like (depends on how the scholars actually do it, but since they will probably differ a lot there needs to be some decision making point in which this is exactly determined):

chunk1 could be “عن حُمْرانَ مَوْلَى عُثْمَانَ بْنِ عَفَانَ”, in which the comments of scholars who regarded this as one chunk (i.e. one ‘stop here and explain’) can be placed. Now, if some scholars stopped at “عن حُمْرانَ” and then at “عُثْمَانَ بْنِ عَفَانَ”, while others stopped only once at the end of “عن حُمْرانَ مَوْلَى عُثْمَانَ بْنِ عَفَانَ”, then the two explanations of the previous group of scholars can just all come under the leaf handling “عن حُمْرانَ مَوْلَى عُثْمَانَ بْنِ عَفَانَ”, because otherwise I fear it would introduce too much complexity.
