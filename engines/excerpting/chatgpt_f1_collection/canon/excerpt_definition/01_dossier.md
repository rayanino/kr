# Purpose and scope

This dossier instantiates the current excerpt-definition canon from five reviewed cases only:

- `ext_39_masala:3`
- `ext_39_masala:6`
- `ext_46_qa:9`
- `ibn_aqil_v1:11`
- `ibn_aqil_v3:13`

It is a constrained backfill pass. Anything not supported by those cases remains provisional, unresolved, or absent.

# Current core definition candidate

Current three-level candidate:

1. **Excerpt candidate**
   - A bounded textual unit proposed as one primary study object, or one tightly bound explanatory cluster, rather than a broad area or casual aggregation.  
   - Current support: `PR-016` (provisional).

2. **Acceptable excerpt**
   - An excerpt candidate that preserves enough frame, function, and referential clarity that the reader does not need essential backward hunting or guesswork just to understand what sort of unit it is and what job it is doing.  
   - Current support: `PR-001` (provisional), `PR-003` (provisional).

3. **High-quality directly study-ready excerpt**
   - An acceptable excerpt that is cleanly enterable, not deceptively cut, not avoidably overloaded, and preserves the relations needed for direct study.  
   - Current support: `PR-002` (accepted, narrow scope), plus provisional pressure from `PR-004`, `PR-007`, `PR-009`, `PR-012`, `PR-015`.

# Identity vs acceptability vs quality

| Level | Current meaning | Current status |
|---|---|---|
| Excerpt candidate | One primary study object or one tightly bound explanatory cluster | provisional |
| Acceptable excerpt | No essential backward hunting for frame/function/referent | provisional |
| High-quality directly study-ready excerpt | Acceptable + clean entry + no deceptive omission + no avoidable overload | provisional with one narrow accepted sub-rule |

Important current split:

- A unit may carry real theory and still fail **direct study-readiness**.
- Reviewed cases repeatedly pressure the distinction between:
  - **independently understandable enough to recover theory**
  - **safe/directly study-ready enough to study from without local hunting**

This distinction is present in the canon now, but its threshold is not fully settled.

# Necessary properties already supported

Supported means: pressured by reviewed cases strongly enough to belong in the dossier now.

1. **Entry-frame clarity matters**  
   Bare list numbers, unexplained conjunctions, unresolved pronouns, ambiguous opening labels, or unexplained entry markers can break direct study-readiness.  
   Support: `PR-002`.

2. **Essential backward hunting is disqualifying at least at the quality level**  
   If the reader must go backward to recover who is being referred to, what list item this is, or what kind of unit they are reading, the unit underperforms.  
   Support: `PR-003`.

3. **Functionally mixed units trigger boundary pressure**  
   Reviewed cases repeatedly show pressure when one unit does more than one primary job without a justified binding relation.  
   Support: `PR-004`, `PR-016`.

4. **Role/layer preservation matters**  
   Quoted material and proof material cannot be classified only by surface content; current-source role can matter.  
   Support: `PR-005`, `PR-015`.

5. **The explained object and its explanation often need default unity**  
   Splitting them can misbind explanation to the wrong object or force hunting.  
   Support: `PR-007`.

6. **Omission honesty matters when cuts are made**  
   A detached unit should not silently look like uninterrupted source flow if meaningful material was omitted.  
   Support: `PR-012`.

# Non-properties / what is not yet required

Not enough evidence exists to require the following as universal doctrine:

- absolute topic purity with zero justified overlap
- hard numeric length caps
- always splitting every sibling mention into a separate excerpt
- always keeping all examples only inside theory units
- always splitting quoted material internally
- always gluing all footnotes
- always re-excerpting all footnotes
- any universal add-on/context-note scheme

# Disqualifying failure conditions already supported

These are supported as real failure patterns in the reviewed cases, but not all are equally frozen in scope.

1. **Opaque entry markers**
   - `2`, `7 - صحيح`, `⌜10⌝`, opening `و`
   - Cases: `CS-001`, `CS-002`, `CS-004`, `CS-005`

2. **Unresolved referential dependence**
   - e.g. pronoun `عليه` with missing antecedent
   - Case: `CS-001`

3. **Quote-layer ambiguity that changes the role of the unit**
   - Case: `CS-003`

4. **Mixed-function rule/proof packaging**
   - Cases: `CS-001`, `CS-002`, `CS-003`

5. **Sibling-topic overmerge**
   - Case: `CS-005`, with pressure from `CS-004`

6. **Overloaded packaging**
   - matn + sharh + example + long editorial footnote in one unit
   - Case: `CS-004`

7. **Silent cutting that hides omission**
   - Case pressure: `CS-005`

# Doctrine by canonical dimension

## self_containment
- Current doctrine:
  - direct study-readiness fails when essential entry markers are unresolved (`PR-002`, accepted narrow scope)
  - self-containment is stricter than mere topical recoverability (`PR-001`, `PR-003`)
- Status:
  - usable, but the exact threshold between acceptable and directly study-ready remains unresolved (`UQ-001`)

## context
- Current doctrine:
  - context includes entry-frame, source-flow, and quote-layer, not just topic label
  - missing context can make a unit recoverable in theory yet weak in actual study use
- Status:
  - provisional; context doctrine is materially pressured but not fully operationalized (`UQ-001`, `UQ-002`)

## boundary
- Current doctrine:
  - mixed rule/proof units create real boundary pressure (`PR-004`)
  - sibling topics normally should not live in one topic-level excerpt (`PR-009`)
  - explained object + explanation often need to remain together (`PR-007`)
- Status:
  - mixed: some strong pressure, but split rules are not frozen (`UQ-003`, `UQ-004`)

## granularity
- Current doctrine:
  - no hard numeric cap is supported
  - burden matters more than raw size
  - mention alone is not enough to justify an excerpt
- Status:
  - weakly grounded and still underdefined (`PR-008`, `PR-014`, `UQ-009`, `UQ-010`)

## function
- Current doctrine:
  - function cannot be read from surface form alone
  - rule/proof distinction is real and operationally important
- Status:
  - emerging, not frozen (`PR-004`, `PR-006`)

## attribution_quote_layer
- Current doctrine:
  - current-source role can override the internal appearance of the quoted block
  - preserving quote-frame is part of relation integrity
- Status:
  - thin but real (`PR-005`, `PR-015`, `UQ-005`)

## rule_vs_proof
- Current doctrine:
  - rule and proof are distinct jobs
  - collapsing them into one unit is often boundary-suspicious
  - quote-based proof handling remains unresolved in harder cases
- Status:
  - emerging (`PR-004`, `PR-015`, `UQ-004`, `UQ-005`)

## theory_vs_example
- Current doctrine:
  - examples can be part of theory exposition
  - examples may also need separate practice-facing placement
  - the frame introducing an example may belong with the example
- Status:
  - promising but still thin (`PR-007`, `PR-011`, `UQ-006`)

## relation_integrity
- Current doctrine:
  - preserving who proves what and who quotes whom matters
  - strategic overlap may sometimes preserve useful relation information
- Status:
  - emerging (`PR-010`, `PR-015`, `UQ-003`, `UQ-005`)

## study_readiness
- Current doctrine:
  - this is stricter than independent understandability
  - accepted narrow rule: unresolved backward-dependent markers break direct study-readiness
- Status:
  - partly stable (`PR-002`), broader threshold unresolved (`UQ-001`)

## omission_honesty
- Current doctrine:
  - excerpt cuts should not silently impersonate uninterrupted source flow
- Status:
  - real but still thin (`PR-012`, `UQ-007`)

## footnotes
- Current doctrine:
  - directly related footnotes should normally stay attached to their anchor text
  - broader footnote doctrine is not settled
- Status:
  - very underdefined (`PR-013`, `UQ-008`)

# Current decision procedure candidate

Current candidate sequence, evidence-limited:

1. **Identify the unit type at entry**
   - If not possible without backward hunting, direct study-readiness fails (`PR-002`).

2. **Ask whether the unit has one primary study object or one tightly bound explanatory cluster**
   - If not, candidate identity is weak (`PR-016`).

3. **Check for essential backward hunting**
   - referent?
   - list position?
   - quote layer?
   - what job the unit is doing?
   - If yes, acceptability is pressured (`PR-003`).

4. **Check for mixed-function packaging**
   - rule/proof
   - theory/example
   - quote/proof/frame
   - footnote/editorial overflow
   - If mixed, boundary doctrine is engaged (`PR-004`, `PR-009`).

5. **Check whether anything that should stay together was split, or anything that should split was glued**
   - explained/explanation
   - example/frame
   - quote/frame
   - footnote/anchor
   - (`PR-007`, `PR-013`, `PR-015`)

6. **Check omission honesty**
   - if the unit depends on omitted material, is that hidden or explicit?
   - (`PR-012`)

7. **Classify the result by level**
   - candidate only
   - acceptable excerpt
   - directly study-ready excerpt

# What would be reckless to freeze right now

It would be reckless to freeze any of the following into final doctrine:

- a universal strategic-overlap rule
- a universal quote-splitting rule
- a universal footnote rule
- a hard rule for when examples require dual placement
- a hard burden metric
- a universal mention-vs-topic threshold
- a universal omission-signaling format
- a fully confident final definition of direct study-readiness
