# Hard Judgment

## What is already strong

1. **Entry-frame failure is real and now well-grounded**
   - Four different reviewed cases support the narrow accepted rule that unresolved backward-dependent entry markers break direct study-readiness.

2. **The canon can now distinguish levels**
   - candidate
   - acceptable excerpt
   - directly study-ready excerpt  
   This distinction is no longer optional.

3. **Mixed rule/proof pressure is real**
   - Three reviewed cases independently force it.

4. **Quote-layer function is now on the board**
   - Only thinly, but decisively: the canon can no longer treat quoted material as flat.

5. **The canon has real non-hollow evidence artifacts**
   - especially the user-reconstructed alternatives for `ext_46_qa:9`, `ibn_aqil_v1:11`, and `ibn_aqil_v3:13`.

## What is still weak

1. **Positive doctrine is thin**
   - Most evidence is failure evidence.
   - The canon knows more about what breaks than what cleanly passes.

2. **Granularity doctrine is still weak**
   - mention vs true topic
   - burden
   - strategic overlap
   These are present, but not hardened.

3. **Example doctrine is still weak**
   - dual placement
   - example anchoring
   - theory-facing vs practice-facing distinction  
   The ideas are strong, but the evidence is still narrow.

4. **Quote-layer doctrine is fragile**
   - one serious case is not enough for a stable canon.

5. **Footnote doctrine is very weak**
   - one case only
   - mixed pressures inside that one case

## What is still dangerously underdefined

1. **The threshold between acceptable and directly study-ready**
2. **The operational rule for strategic overlap**
3. **The split protocol for mixed rule/proof units**
4. **How quote-frame survives splitting**
5. **How omission should be signaled honestly**
6. **When examples require dual placement**
7. **How to distinguish mention from true topic**
8. **How to handle footnotes beyond directly attached notes**

## Highest-value future excerpt types for hardening

1. **A genuinely strong positive passing excerpt**
   - needed to stop the canon from being failure-only

2. **A long rule with multiple linked proofs**
   - needed for rule/proof doctrine and relation integrity

3. **A long quotation used as proof inside another source**
   - needed for quote splitting and quote-frame preservation

4. **A clean matn/sharh case that clearly works**
   - needed to test whether current matn/sharh pressures are overfitted to one overloaded case

5. **A footnote-heavy case with clearly separable footnote types**
   - needed for any serious footnote doctrine

6. **A case where strategic overlap clearly helps**
   - paired with one where it clearly harms

7. **A case with examples embedded in theory and also grouped later for practice**
   - needed to harden dual-placement doctrine

## What would be reckless to freeze into final doctrine right now

It would be reckless to freeze:

- a final universal definition of self-containment
- a final split/merge doctrine
- a final quote-layer doctrine
- a final example-placement doctrine
- a final footnote doctrine
- a final omission-honesty mechanism
- a hard burden rule
- a universal mention-vs-topic threshold

## Bottom line

The canon is now real, not just commentary. But it is still a **pressure-tested early canon**, not a final one.

What is strongest right now:
- entry-frame doctrine
- level distinction
- boundary pressure around mixed rule/proof packaging

What is most dangerous right now:
- mistaking repeated pressure for completed doctrine
- pretending that thin positive evidence is enough
- freezing strategic overlap, quote-layer handling, example placement, or footnote handling too early
