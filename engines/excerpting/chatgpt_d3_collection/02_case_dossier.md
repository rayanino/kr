## 3. Engineering / protocol expansion

### 3.1 The menu is flattening

The menu asks about three kinds of definition as if they were obviously flat peers. The raw owner reaction does not support that flattening.

More faithful structure:

- **linguistic definition**: own unit
- **technical definition**: own unit
- **school-specific definition**:
  - separate if genuinely distinct
  - branch under technical if it is a school-specific refinement / narrower definition
  - attribution only if it is merely “this school adopts the same technical definition”

### 3.2 Packaging is not ontology

The raw reaction supports allowing short harmless carry-over in some cases. But that does **not** change the ontology.

Examples in this case:
- a short proof-related phrase may remain inside a definition excerpt for current-case packaging reasons
- an attribution excerpt may temporarily carry short proof/definition material when that whole stretch is short and tightly coupled

That does **not** mean “definition, proof, and attribution are one excerpt-type.”

### 3.3 Leaf pollution pressure

A quick proof mention that is later treated in fuller form may lose significance as a leaf-worthy unit. This introduces a serious “leaf pollution” pressure:

- not every mention deserves its own leaf
- some mentions are only support or orientation inside another unit
- significance must be evaluated before giving text its own tree location

This remains real but not fully settled from one case.

### 3.4 Source hints are useful but non-deciding

The raw owner reaction strongly preserves this distinction:

- punctuation, commas, author flow, sequencing, layout, references, and diacritics are useful **hints**
- they may improve confidence
- they must **not** become deciding authority for excerpt structure

### 3.5 Pre-excerpt analysis / security gates

This case reinforces a broader requirement:

- perform deeper passage analysis before excerpting
- gather structural hints and pattern signals
- use them as non-deciding confirmation
- do not let them replace the core excerpting logic

## 4. What is certain
