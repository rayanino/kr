# IntakeDossier

Owned by:

- `40-intake-analysis`

Purpose:

- Preserve the evidence packet that metadata deliberation must consume.

Core fields:

- `dossier_id`
- `source_format`
- `normalization_route`
- `title_evidence`
- `work_identity_proposal`
- `collection_match_candidates`
- `declared_vs_observed_counts`
- `completeness_status`
- `partiality_reasons`
- `integrity_status`
- `integrity_findings`
- `pdf_text_layer_status`
- `pdf_text_evidence`
- `page_layout_hint`

Rules:

- Work identity is proposed here, not finalized here.
- Completeness and integrity verdicts are mandatory dossier outputs for every non-blocked upload.
